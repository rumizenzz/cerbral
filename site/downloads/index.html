Cerbral for Mac — almost there.

Cerbral installs as a normal Mac app: double-click the .dmg, drag Cerbral.app into Applications, double-click Cerbral to launch. The bundle lays out your Mac the same way the desktop flow does:

  • Cerbral.app — the desktop client.
  • cloudflared tunnel — bundled as a sidecar, so Cerbral Web on cerbral.com can reach the AI running on your Mac.
  • Ollama — install separately from ollama.com. Cerbral prompts you on first run if it's missing.
  • Your AI model — pulled by Cerbral on first run. Default is Qwen3:8B (~5 GB); we'll recommend a bigger model if your Mac can handle it.

Coming soon: the .dmg will live at github.com/rumizenzz/cerbral-desktop/releases/latest/download/Cerbral.dmg and /downloads/Cerbral.dmg on cerbral.com will redirect there automatically. Until it's uploaded, you're on this page.

Building it yourself? Clone rumizenzz/cerbral-desktop and run make dmg. You'll get the same .dmg in src-tauri/target/release/bundle/dmg/.
