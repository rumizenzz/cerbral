Cerbral for Mac — almost there.

Cerbral installs as a normal Mac app: double-click the .dmg, drag Cerbral.app into Applications, double-click Cerbral to launch. The bundle lays out your Mac the same way the desktop flow does:

  • Cerbral.app — the desktop client.
  • cloudflared tunnel — bundled as a sidecar, so Cerbral Web on cerbral.com can reach the AI running on your Mac.
  • Ollama — install separately from ollama.com. Cerbral prompts you on first run if it's missing.
  • Your AI model — pulled by Cerbral on first run. Default is Qwen3:8B (~5 GB); we'll recommend a bigger model if your Mac can handle it.

The .dmg lives at github.com/rumizenzz/cerbral-releases/releases/latest/download/Cerbral.dmg, and /downloads/Cerbral.dmg on cerbral.com redirects there automatically. Every new version updates that link with no site-side change.

If macOS says "Cerbral OS Not Opened"

macOS's Gatekeeper blocks apps that aren't signed through the paid Apple Developer Program. Cerbral is legitimate — we just haven't registered yet. Here's the one-time unblock:

  1. If you see the "Not Opened" dialog, click Done.
  2. Make sure Cerbral OS is in your Applications folder.
  3. Open Terminal and run (one line, paste-able):
sudo xattr -dr com.apple.quarantine "/Applications/Cerbral OS.app"

Enter your Mac password when asked. Then double-click Cerbral OS from Applications and it will launch normally. You only need to do this once per install — updates replace the app in place.

Why this is needed: Apple charges $99/year for the developer certificate that removes this prompt. Galactuz will register once the Cerbral Cloud subscription revenue earns it. Until then, every .dmg we ship is unsigned but otherwise identical to what a signed build would be.
